{#- Boilerplate - User Macros ------------------------------------------------------------ This file contains macros for user related tasks. -#} {#- Macro: avatar ------------------------------------------------------------ Dieses Makro erzeugt einen runden Initialen-Badge lokal (kein externer Request). Parameter: first_name, last_name, bg_color (hex ohne #), text_color (hex ohne #), size (Zahl in px, wird für Schriftgröße genutzt), class (zusätzliche CSS-Klassen). -#} {% macro avatar(first_name, last_name, bg_color, text_color, size, class) %} {{ first_name[0] | upper }}{{ last_name[0] | upper }} {% endmacro %}
